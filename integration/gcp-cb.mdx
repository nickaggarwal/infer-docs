### Preparing the repository 

To import via Google Cloud buckets method you need to create a zip file with the following structure. 

```bash
./
├── model.<extension>
├── input.json
├── output.json
├── config.pbtxt ( Optional )
```


We have creted a sample Google Cloud Storage URI that you can use to test the inferless platform.

File Link: [https://github.com/infer-less/template-method](https://github.com/infer-less/template-method)

Once you have the zip file ready you can follow the below steps to import the model.


### Steps to import model zip 

1. Go to the [Inferless Dashboard](https://app.inferless.com) and click on the **Add Model** tab.
2. Select the Google Cloud Storage option and click on the **Next** button.
3. Enter the Google Cloud Storage URI for the model.
4. Select the name of the model. 
5. Select the machine configuration. 
6. Optionally you can configure custom runtime, by default will use inferless default runtime.
7. Optionally you can also mount volumes for the model.
