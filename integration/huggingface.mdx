### Supported Frameworks

You can use a `transformer` or a `diffuser` the based model from Hugging face.


### Connect the Hugging Face Account

1. Go to the [Inferless Dashboard](https://app.inferless.com) and click on the **Integrations** tab.
2. Select the Huggingface option and click on the **Connect** button.
3. You can enter name and enter the read token for Inferless
4. Click on **Save** to connect Huggingface Account. 
5. You will be redirected back to the Inferless dashboard.


### Pre Requisite: Note the Model Name, Type, and Framework

* Navigate to the Hugging Face model page of your choice that you want to import into Inferless.

* Take note of the `"Model Name" `(you can also use the copy button), `Task Type`, `Model Framework,` and `Model Type`. These will be required for the next steps.

<Frame caption = "The fields to be copied/noted are mentioned in RED.">![](/images/hugging1.png)</Frame>


### Steps to load your model

To import a model from Hugging Face, below are the requirements.


### Importing the model via Dashbaord 


1. Go to the [Inferless Dashboard](https://app.inferless.com) and click on the **Models** tab.
2. Click on the **Add Model** button.
3. Select the **Huggingface** option and the account connected.
4. Enter the **Model Name** , **Model Type** and **Task Type**.
5. Select the machine configuration. 
6. Optionally you can configure custom runtime, by default will use inferless default runtime.
7. Optionally you can also mount volumes for the model.
